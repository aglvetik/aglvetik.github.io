RISE

Концепция модерационного бота
Общее

Префиксный модерационный бот с упором на простоту, скорость и контроль. Работа через команды без slash-интерфейса.

Команды
!мут / !м @пользователь 10 м причина
!кик / !к @пользователь причина
!бан / !б @пользователь причина
!очистить 20
Формат времени
с — секунды
м — минуты
ч — часы
д — дни

Пример:

!м @user 1 ч флуд
Указание пользователя

Поддерживается несколько форматов:

@mention
ID пользователя
ник (если уникален)
Логика работы

— муты с таймером и авто-снятием
— проверка ролей (иерархия)
— запрет наказаний администраторов
— защита от самонаказания
— обработка некорректных команд

Очистка сообщений

Удаление последних сообщений в канале с ограничением по количеству.

!очистить 20
Логирование

Фиксация всех действий в отдельный канал:

модератор → пользователь → действие → причина → время
Расширение

— система предупреждений
— авто-наказания
— авто-модерация
— настройка ролей

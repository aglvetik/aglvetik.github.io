RISE

Концепция модерационного бота для Discord
Префиксные команды
!мут (!м)
!кик (!к)
!бан (!б)
!очистить [кол-во]
Формат работы команд

Указание пользователя (только для Discord):

  • @user
  • ID (123456789)
  • ответ на сообщение пользователя

Примеры:

!мут @user 10 м причина
!м 123456789 1 ч спам
!бан @user причина
!кик @user причина
!очистить 20
Логика мута
  • поддержка времени: с / м / ч / д
  • временный мут снимается автоматически
  • при муте отключается возможность писать (роль mute в Discord)
  • если роль отсутствует — создаётся или используется указанная
Очистка сообщений
  • !очистить N — удаляет последние сообщения в канале Discord
  • ограничение на количество (например до 100)
  • игнор старых/системных сообщений (ограничения Discord API)
Права и проверки
  • нельзя применять команды к равным/высшим ролям
  • нельзя трогать администраторов
  • проверка прав модератора
  • защита от самонаказания
Логирование
  • отдельный лог-канал Discord
  • фиксируется:
    • кто выдал
    • кому
    • команда
    • причина
    • время (для мутов)
  • формат: embed или текст
Обработка ошибок
  • неверный формат → пример команды
  • пользователь не найден → ошибка
  • нет прав → отказ
  • некорректное время → ошибка
Дополнительно
  • !варн — система предупреждений
  • авто-наказания за накопление
  • авто-модерация (фильтр слов/спама)
  • настройка префикса
  • настройка ролей (захардкожено)
